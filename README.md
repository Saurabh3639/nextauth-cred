## Technolgy Stack
Next JS + Auth.js + Prisma + MongoDB Atlas

## Getting Started
```bash
cd nextauth-cred
npx create-next-app@latest .
npm run dev
```

## Dependencies
```bash
npm i next-auth bcrypt
npm i @prisma/client @auth/prisma-adapter
npm i prisma --save-dev
```
